NBA Salary Efficiency Dashboard (2024-25)

Project Summary

This dashboard merges NBA player performance and salary data to evaluate contract value.

Players were filtered to rotation-level contributors: at least 40 games, at least 20 MPG, and salary at least $0.5M.

Value is measured using Efficiency Surplus = Actual PER - Expected PER, where Expected PER is predicted from salary using a polynomial regression model.
